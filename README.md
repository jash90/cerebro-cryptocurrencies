# cerebro-cryptocurrencies

> [Cerebro](http://www.cerebroapp.com) plugin for convert cryptocurrencies

![](screenshot.png)

## Usage

Just enter what you want!

## Features
* Supports cryptocurrencies;
* Understands currency codes;
* Supports English `500 eth in pln`
